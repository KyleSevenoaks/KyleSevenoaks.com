The CSS3 slider does exactly what it's name suggests. It slides things and is built entirely out of normal HTML5 and CSS3. No javascript required!

You can put anything within the slide, links, text, images, images with hidden captions (that have more images), iFrames.. The list is endless.

You can have any number of slides, and set any one of them to be the default slide (the one you see first.)

This is completey JS free. This demo uses jQuery to add a class to the slider to switch between manual and automatic modes. For demonstration purposes only.
